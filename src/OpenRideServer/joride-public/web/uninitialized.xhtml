Uninitialized Properties

If You can see this page, then something is wrong with the setup of joride:

At least one of the properties

  • imprintURL
  • termsURL
Is not set properly.

If You find this message on the internet, please inform the administrator of this site. If You find this message on a local intranet, or on Your local host, then there is no need for action.

To fix the problem, an administrator must adjust the imprintURL and/or termsURL properties in the local.properties file, then rebuild and deploy OpenRide.
